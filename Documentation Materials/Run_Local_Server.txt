If You Have Node.js Installed:
Install http-server:

Open your terminal or command prompt.

Navigate to your project directory using "cd c:\Users\jarro\OneDrive\Desktop\CaseRace\Building MVP"

Run "npm install --global http-server" to install a simple, zero-configuration command-line HTTP server globally on your machine.

Start the Server:

In your terminal, still within your project directory, run "http-server".

This command serves your project on a local server, typically available at http://localhost:8080.
